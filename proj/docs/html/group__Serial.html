enum SerialCommand

Enumeration of all supported serial commands for Delta Sync.

This protocol avoids sending the entire file on every change. Instead, it transmits precise "Deltas" (insertions, deletions, and block replacements).

Enumerator
CMD_INSERT_CHAR 

Insert a character. Payload: 1 byte.

CMD_DELETE_CHAR 

Delete char before cursor. Payload: 0 bytes.

CMD_MOVE_CURSOR 

Move remote cursor. Payload: 4 bytes (2 row, 2 col).

CMD_REPLACE_BLOCK 

Prepare memory for block edit. Payload: 6 bytes (2 start_row, rows_deleted, rows_inserted).

CMD_UPDATE_LINE 

Overwrite a line. Payload: 2 bytes (row_index) + text (max 250 char).

CMD_FILE_START 

Start file share. Payload: 2 bytes (Numlines) + filename.

CMD_FILE_LINE 

Send file line during sync. Payload: text (max 252 char).

void serial_process ( )

Processes incoming bytes from the serial port.

Implements a state machine (WAIT_START -> READ_CMD -> READ_LEN -> READ_PAYLOAD) to assemble bytes into SerialEvent structs and pushes them to the input queue.
